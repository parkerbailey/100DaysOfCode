# Coding Notes

## HTML5
### Anchors
Creates hyperlink between *source anchor* and *destination anchor*
e.g. `<a>an anchor</a>`
Hyperlink reference:    `href=""`
    # = dead link
Target:                 `target=""`
    _blank = opens in new window

### Unordered List
Creates a bulleted list
e.g.
```
<ul>
    <li>item 1</li>
    <li>item 2</li>
</ul>
```

### Ordered List
Creates a numbered list
e.g.
```
<ol>
    <li>item 1</li>
    <li>item 2</li>
</ol>
```
